# Fast start
``` shell
	# Terminal 1
	git clone https://github.com/dbwodlf3/uidoc.git
	cd uidoc
	npm i
	cd src/server
	npm i
	cd ../client
	npm i
	cd ../../
	npm run start
```
``` shell
	# Terminal 2
	cd uidoc
	npm run watch-client
```
